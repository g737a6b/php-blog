## Installation

### Composer

Add a dependency to your project's `composer.json` file.

```json
{
	"require": {
		"g737a6b/php-blog": "*"
	}
}
```

## Development

### Run tests

```sh
docker run -it --rm -v $(pwd):/app composer:1.8 run-script tests
```
